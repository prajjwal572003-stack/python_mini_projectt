# 📊 Value Prediction Project (Machine Learning)

This project is a simple Machine Learning model that predicts future values using **Linear Regression**.

---

## 🚀 Features
- Train model using given input and output data
- Predict future values
- Display graph using matplotlib
- Calculate performance metrics:
  - R2 Score
  - Mean Absolute Error
  - Slope
  - Intercept

---

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## 📂 Project Structure

python-mini-project/
│
├── main.py                # Prediction class
├── value_prediction.py    # Main ML logic
├── requirements.txt       # Dependencies
└── README.md

---

## ▶️ How to Run

### 1. Install dependencies

pip install -r requirements.txt
### 2.  Run The Project
python value_prediction.py


📈 Example

Input:
Years = [1, 2, 3, 4, 5, 6]
Earnings = [1000, 800, 2000, 1500, 3400, 4000]

Output:
Predicted future value using Linear Regression
